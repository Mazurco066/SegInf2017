 **[escrito por]** mazurco066 (Gabriel Ribeiro)

## seginf2017_bruteforce

 **[bruteforce.sh]** shellscript

 **[plataforma]** Linux
 
 **[Disponibilização]** Use chmod 777 bruteforce.sh para poder usa-lo
